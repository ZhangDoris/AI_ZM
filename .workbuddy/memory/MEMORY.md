# 项目长期记忆

## 项目概述
智能预筛系统 — 临床试验受试者预筛选管理平台

## 技术选型
- 原型：纯 HTML + CSS + 原生 JS（Ant Design 视觉风格，无框架依赖）
- 后续开发：React + Ant Design（用户需求）

## 页面结构
- 顶部横向一级菜单：审核列表 / 智能筛选 / 项目管理 / 数据统计 / 项目中心邀请码 / AI改进分析 / 后台管理 / 脱敏管理 / 测试后台
- 数据统计、AI改进分析、后台管理：有左侧二级菜单
- 其余模块（含项目管理）：仅单页列表

## 设计风格
- 深色顶导航（#141c2d）
- 白色内容卡片 + 浅灰背景（#f5f7fa）
- 主色调：蓝色（#1677ff，Ant Design 5.x 主题色）
- 现代简洁风格，圆角卡片、徽标状态标签
