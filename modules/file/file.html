文件夹显示配置说明
  • 固定项（不可关闭）：原始资料、审批上传附件存档 — 系统核心文件夹，始终显示
  • 可配置项（开关控制）：基因检测报告、导入期资料等 — 可通过开关控制显示/隐藏
  • 关联依赖项：基因检测报告受「项目基因检测」功能影响，关闭时需二次确认
  • 提示语配置：点击编辑按钮可配置进入该文件夹后显示的提示内容
  • 子目录管理：点击「新建」可在对应文件夹下创建一级/二级子目录。注意：AI分类资料、审批上传附件存档、AI预筛结果存档、预筛表提取结果存档不支持新建子目录（系统托管/自动管理）
  • 拖拽排序：拖拽每行右侧图标可调整文件夹在侧栏中的显示顺序
  • 操作历史：每行的时钟图标可查看该开关的操作痕迹记录
