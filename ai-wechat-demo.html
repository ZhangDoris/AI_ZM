📋 使用说明

  1. 直接打开：双击 ai-wechat-demo.html 文件在浏览器中打开
  2. 查看企业微信报告：点击"查看企业微信报告"按钮体验适配页面
  3. 查看完整报告：点击"查看完整报告"按钮查看原始AI分析报告
  4. URL参数：支持通过URL参数传递项目ID，如 ?pid=SHR-1918-302
  5. 移动端适配：页面已针对企业微信移动端优化

🔧 技术实现

// 获取URL参数示例 function getUrlParams() { const params = {}; const queryString = window.location.search.substring(1); const pairs = queryString.split('&'); for (const pair of pairs) { const [key, value] = pair.split('='); if (key && value) { params[decodeURIComponent(key)] = decodeURIComponent(value); } } return params; } // 动态更新页面内容 function updatePageForProject(projectId) { // 根据项目ID更新标题、数据等 const titleElement = document.querySelector('.report-title'); if (titleElement) { titleElement.innerHTML = `📊 ${projectId} · AI分析报告`; } }
